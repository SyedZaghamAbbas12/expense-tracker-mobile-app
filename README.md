💰 My Expenses - Mobile App

A simple Expense Tracker mobile application built using React Native and Expo CLI.

📱 Features

- Add new expenses (Title, Amount, Category)
- Input validation (no empty title, positive amount, category required)
- View list of all expenses
- Delete expenses from list
- Live total calculation
- Category-based organization

🛠 Tech Stack

- React Native
- Expo CLI
- JavaScript

🚀 How to Run the Project

1. Install dependencies
```bash
npm install
Start the Expo server
npx expo start
Run on Android
Scan QR code using Expo Go app
OR
Run Android Emulator
